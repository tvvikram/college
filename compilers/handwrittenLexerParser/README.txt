
Handwritten Lexer and Parser for our subset of C language

Commands:
a. cd to extracted directory.
b. use `make clean' once.
c. then use `make' to build the executable.
d. name your test cases as test*.cs
e. test it as `make test ARGS=*', * is a number > 2
f. do send the test cases, and inform the problems.
