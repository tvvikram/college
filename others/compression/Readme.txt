Both Compress and decompress has been merged into same file. 

For Compression without dotty output , Run
	mvh -c <Source File> <Destination File>

For Compression with dotty output , Run
	mvh -c <Source File> <Destination File> --dotty
	
For Decompression, Run
	mvh -d <Source File> <Destination File>
	
Or Use the Shell script to Run after Make
